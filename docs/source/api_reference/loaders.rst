Loaders Module
==============

The ``nlp_utilities.loaders`` module provides utilities for loading language-specific data and configurations.

Data Loading
~~~~~~~~~~~~

These functions load various types of linguistic data needed for validation, normalization, and conversion:

- :func:`load_language_data`: Load language-specific feature definitions
- :func:`load_whitespace_exceptions`: Load exceptions for whitespace handling
